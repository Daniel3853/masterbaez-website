# Cómo usar el botón EDITAR y subir fotos/videos

## Botón "✏️ EDITAR" — ahora siempre visible
El botón morado aparece siempre en la esquina inferior derecha del sitio.
Haz clic para activar el modo edición y poder:
- **Editar textos**: haz clic directo sobre cualquier texto y escríbelo
- **Subir fotos/videos**: verás botones "📷" en cada espacio vacío

## Subir fotos y videos al sitio
1. Haz clic en "✏️ EDITAR" para activar el modo edición
2. Busca los espacios con el texto "+ Add photo" o "+ Add photo/video"
3. Haz clic y selecciona el archivo de tu computadora
4. La foto/video se verá al instante (solo tú la ves por ahora)
5. Cuando estés listo para publicar, sube la misma foto desde /admin

## Programa popup — video y checkout
En la sección de Programas, al hacer clic en "Learn More":
1. Se abre un popup con espacio para tu video/foto del programa
2. Puedes subir una foto/video directamente desde el popup
3. Haz clic en "Continuar" para ir al formulario
4. Después del formulario, pega la URL de tu página de checkout/pago
   y haz clic en "Ir a Checkout →"

## Configurar URLs de video y checkout
Para configurar cada programa (video, link y checkout):
1. Abre `index.html` con un editor de texto (Bloc de Notas)
2. Busca: `CONFIGURACIÓN DE CADA PROGRAMA`
3. Ahí puedes poner las URLs directas para cada programa

Opciones:
- `video`: URL del video (YouTube o .mp4)
- `link`: URL de tu lead page (opcional, salta el formulario)
- `checkout`: URL de pago después del formulario
- `imagen`: URL de la imagen del programa

## Blog — fotos en los artículos
1. Abre el Editor de Páginas desde la carpeta del proyecto:
   `editor-blog-master-baez.html`
2. Arrastra el archivo .html del blog al editor
3. Usa los botones "🖼️ Subir Foto" o "🖼️ Foto por URL"
4. Edita el texto directamente en la página
5. Descarga los cambios y súbelos a GitHub

## Publicar los cambios
Cuando termines de editar:
1. Haz clic en "💾 Guardar borrador" (guarda en tu navegador)
2. Haz clic en "⬇️ Descargar content.json"
3. Sube ese archivo a GitHub en la carpeta `/data/content.json`
4. En 1-2 minutos los cambios se verán en el sitio para todos

## También puedes editar desde /admin
Entra a `tusitio.com/admin` para editar textos y subir fotos/videos
directamente sin descargar archivos.
